## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

Move into the project

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Open

```
http://localhost:3000
```
